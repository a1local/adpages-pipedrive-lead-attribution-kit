# Publish Blockers

This folder is ready as a local resource kit, not as a Pipedrive Marketplace submission.

## Must Be Done Before Public Marketplace Claims

- Decide whether the public asset is a resource/template, a partner guide, or a real Pipedrive Marketplace app.
- If building a real app, add OAuth, scopes, callback handling, uninstall handling, support processes, hosted infrastructure, and app review metadata.
- Create production screenshots, icon artwork, listing copy, support URL, terms URL, and hosted privacy URL.
- Validate all custom field labels and pipeline/stage assumptions against a real Pipedrive account.
- Add account-specific instructions for replacing `manual_*` placeholder field keys.
- Complete security and privacy review for any future credential handling, automatic sync, API write-back, analytics, or hosted storage.

## Current Non-Goals

- No Pipedrive API client.
- No OAuth flow.
- No marketplace app manifest.
- No webhook receiver.
- No hosted backend.
- No automatic lead, deal, person, or organization writes.
- No analytics or tracking.
